# sprintr
The folder contains the code for reproducing the numerical studies in the paper, "Reluctant Interaction Modeling in Generalized Linear Models".

The package `sprintr` contains the implementation of the proposed method. To install `sprintr` from GitHub, enter the following command in the R console:

```r
devtools::install_github("hluscience/sprintr")
```

Note that the installation above requires using R package [devtools](https://cran.r-project.org/web/packages/devtools/index.html), which can be installed using `install.packages("devtools")`.
